# moga-component-build-plugin

## 介绍
基于render-builder封装的编译react组件的build-plugin

## 用法

在根目录的中配置文件中（这里以build.json为例）添加如下代码即可：

```
export default {
    plugins: ['moga-component-build-plugin']
}
```
